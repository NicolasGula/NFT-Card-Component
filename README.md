### Que aprendi

- Utilizar Flexbox para alinear elementos con align-items y justify-content



```css
footer{
    display: flex;
    align-items: center;
    margin-top: 12px;
    font-size: 1em;
}

.items{
    display: flex;
    flex-wrap: wrap;
    align-items: center;
    justify-content: space-between;
    margin: 18px 0;
    font-size: 15px;
}
```
